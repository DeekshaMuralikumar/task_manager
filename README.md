Task Manager (Full Stack)

A simple Full Stack Task Manager Application built using Spring Boot, React, H2 Database, JUnit, and AOP.

This project demonstrates basic CRUD operations, unit testing, logging using Aspect-Oriented Programming, and in-memory database usage.

Technologies Used

Backend

Spring Boot
Spring Data JPA
H2 Database
Spring AOP
JUnit

Frontend

React
Axios
CSS
Features
Add task
View tasks
Delete task
Logging using AOP
Unit testing using JUnit
In-memory database using H2

JUnit:

Tests are written for the service layer to verify task operations.

Run tests using:
mvn test

AOP:

AOP is used for logging service method execution.
It prints logs before and after service methods run.
